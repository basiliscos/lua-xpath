Overview

Based on LuaExpat, this module is an XPath implementation in Lua, similar to TinyXPath.

lua-xpath is free software and uses the same license as Lua 5.1.

Status

Current version is 1.3. It was developed for Lua 5.1 based on LuaExpat but also should work with Lua 5.2, 5.3 and 5.4.

Download

lua-xpath source can be downloaded from its Github page.

History
